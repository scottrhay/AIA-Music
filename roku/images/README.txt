Image Assets for AIA Music Roku App

These images are OPTIONAL for V1 MVP. The app works without them.

If you want custom branding later, create these files:

1. icon_focus_hd.png (336x210 pixels)
   - App icon when focused on home screen
   - AIA Music logo with blue accent

2. icon_side_hd.png (108x69 pixels)
   - Small icon for Roku UI
   - AIA Music logo simplified

3. splash_hd.png (1280x720 pixels)
   - Loading screen shown on app launch
   - AIA Music branding + "Loading..."

For now, Roku will use default placeholders.

To add images:
1. Create PNG files with names above
2. Place in this folder (images/)
3. Re-package with package.ps1
4. Re-sideload to Roku
